# Epic: EPIC-260072 — Node Visual Design: LOD & Blender-style blocks

- Epic ID: EPIC-260072
- Title: Node Visual Design — Level-of-Detail & Blender-style blocks
- Persona/Actor: Graph users navigating dense workspaces.
- Value statement: For graph users, deliver zoom-progressive node rendering with a dense, organized "Blender-feel" block so the graph stays readable when zoomed out and information-rich when zoomed in.
- Context: `AtomNode` is sparse *and* its type label is too big — the worst of both. The "Blender instinct" from §5: nodes should reveal detail by zoom, like a node editor.
- Problem statement: Nodes don't scale detail with zoom; there is no dense, legible block at high zoom and no calm overview at low zoom.
- Expected outcome: **Level-of-detail rendering** — zoomed out: a colored dot + short title; zoomed in: a dense, organized block (tiny title; a metadata row of type badge / computed value / status icon; label color-dots; category-dimension dots), small/tabular/monospace for values — "the Blender feel." Uses React Flow's zoom signal and the EPIC-260066 node-extension seam, composing cleanly with compute badges (EPIC-260069).
- Success metric(s): Nodes show progressively more detail as the user zooms in; the high-zoom block is dense and legible; low-zoom stays calm.
- Non-goals: The compute status badges themselves (supplied by EPIC-260069; this epic hosts them via the seam); node sockets/ports (not in scope — this is visual density, not a node-wiring redesign).
- Constraints: Branding (REQ-CR-260011) — restrained palette, never color alone; render via the node-extension seam so badges and LOD layers don't collide.
- Dependencies: EPIC-260066 (node-extension seam).
- Risks/assumptions: Both Flow and Network node components may need the LOD treatment; the seam keeps this independent of EPIC-260069.
- Target window: TBD (Ideation).
- Validation approach: Unit tests for zoom-threshold rendering states; visual review against the "Blender feel" target.

- [ ] Dense block layout: monospace/tabular values, label color-dots, category-dimension dots.
- [ ] Composition with EPIC-260069 compute badges via the node-extension seam.
