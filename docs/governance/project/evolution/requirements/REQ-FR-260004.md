# REQ-FR-260004: Label Search and Graph Inspection Experience

- Class: FR
- Status: Accepted
- Date: 2026-04-13
- Source decision: A4 (MVP boundary)

## Requirement

The frontend shall allow users to search atoms by label, inspect atom details from a selection-triggered detail panel, and navigate relationship visualization with zoom and pan.

## Rationale

Search and inspection are required to keep the graph usable as datasets grow and to preserve transparency during editing tasks.

## Scope Notes

- In scope for MVP: label-only search, selection-driven detail panel, zoom/pan graph navigation.
- Out of scope for this requirement: advanced filtering, saved views, and multiple graph presentation modes.

## Verification

- End-to-end flow demonstrates finding an atom by label and focusing it in the graph.
- End-to-end flow demonstrates editing selected-atom fields from the detail panel.
- Usability checks confirm zoom/pan keeps graph exploration practical for MVP dataset sizes.
