# Requirement REQ-OR-260016: Authentication & Authorization Backend Contract (Schema 8.1.0)

- **Status**: Proposed
- **Description**: The frontend MUST integrate against the backend GraphQL schema **8.1.0** auth/authz
  contract: `beginSignup`/`completeSignup`, session-minting `signin`/`signinGoogle`/`completeSignup`,
  `logout`/`revokeAllSessions`, `requestPasswordReset`/`confirmPasswordReset`,
  `requestEmailChange`/`confirmEmailChange`, `linkGoogle`/`unlinkAuthMethod`/`setPassword`, `me`,
  `deleteMyAccount`; Bearer-gated reads; and error codes carried in the GraphQL `errors[].message`.
- **Rationale**: 8.1.0 removed the old `signup` and permanent-token model; the client must consume the
  new operations and error transport to function and to unblock sign-up.
