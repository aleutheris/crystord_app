# REQ-FR-260046: Flow View Edge and Handle Visual Grammar

**Status**: Active
**Type**: Functional Requirement
**Module**: workspace-graph
**Affects**: Flow-view edge rendering, label visibility, and handle placement
**Related ADRs**: ADR-260039, ADR-260040
**Related Requirements**: REQ-FR-260047, REQ-FR-260033

## Summary

Flow view shall use directional edge and connector semantics aligned with left-to-right flow readability.

## Specification

1. Directed edges in Flow view render with visible arrowheads.
2. Bond labels are not rendered in Flow view in this batch.
3. Label data seam remains intact for potential future re-enable.
4. Flow nodes use left/right handles only in MVP.
5. Left/right handle rule is universal in Flow view for this batch (not edge-type dependent).
6. Existing Flow node shape remains unchanged in this batch.

## Rationale

Directional visuals and side-handle conventions improve immediate readability of horizontal flow structures.

## Out of Scope

- New node-shape redesign.
- Additional selected-edge styling changes beyond arrowhead and label suppression decisions.
