# REQ-CR-260005: Search Feature Deferral Boundaries

- Class: CR
- Status: Active
- Date: 2026-04-17
- Source decision: E3, E5 (Autocomplete behavior and saved views/queries)

## Requirement

MVP delivery shall exclude query-builder authoring, semantic multi-field autocomplete, and full saved workspace views; post-MVP sequencing shall prioritize label autocomplete and saved-query support before broader advanced-search capabilities.

## Rationale

Clear deferral boundaries preserve MVP delivery focus while preserving a staged path for search capability growth.

## Scope Notes

- In scope for MVP constraint: exclusion of advanced search capabilities and explicit post-MVP sequencing intent.
- Out of scope for this requirement: detailed implementation design for post-MVP features.

## Verification

- Release checks confirm excluded advanced-search features are not MVP acceptance criteria.
- Backlog checks confirm post-MVP plan includes label autocomplete and saved queries.
- Scope checks confirm saved workspace-view persistence is not included in MVP.
