# REQ-FR-260003: Core Atom and Bond Manipulation in Graph Workspace

- Class: FR
- Status: Active
- Date: 2026-04-13
- Source decision: A4 (MVP boundary)

## Requirement

The frontend shall provide one graph workspace where users can create atoms, edit atom features, and create bonds by dragging an arrow between atoms.

## Rationale

Direct manipulation of atoms and bonds is the core operational behavior that makes the product useful for data-management tasks.

## Scope Notes

- In scope for MVP: atom create/edit, bond create through drag-to-connect, selection of graph elements.
- Out of scope for this requirement: advanced categorization and alternative graph-mode semantics.

## Verification

- End-to-end flow demonstrates creating atoms and connecting them with bonds in the graph.
- End-to-end flow demonstrates editing atom features after selection.
- Contract checks verify backend mutations succeed for atom and bond operations.
