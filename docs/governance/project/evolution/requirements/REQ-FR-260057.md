# Requirement REQ-FR-260057: User Sign-In with Username or Email

- **Status**: Proposed
- **Description**: The system MUST allow a user to sign in using either their registered username or their email address, in combination with their password.
- **Rationale**: Provides flexibility for users who may remember one identifier more easily than the other. This is a standard feature in modern authentication systems.
- **Related ADRs**: [ADR-260049](../adr/superseded/ADR-260049.md)
