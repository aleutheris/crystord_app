# REQ-FR-260006: Read-Only Label Querying and Metadata Inspection

- Class: FR
- Status: Accepted
- Date: 2026-04-13
- Source decision: B2 (Data model coverage)

## Requirement

The frontend shall query existing labels for filtering and display but shall not support label creation or bulk-label administration in MVP.

## Rationale

Label querying enables users to filter atoms and inspect domain structure; label creation is deferred to post-MVP to reduce initial scope.

## Scope Notes

- In scope for MVP: retrieve atoms by label, display available labels in search UI, suggest labels during atom detail editing.
- Out of scope for this requirement: creating new labels, deleting labels, bulk label operations.

## Verification

- End-to-end test demonstrates retrieving atoms filtered by label.
- End-to-end test demonstrates label suggestions in query/filter UI.
- Scope check confirms label creation endpoints are not exposed in MVP UI.
