# Requirement: REQ-FR-260060 - Passwordless Google Sign-In

- **Date**: 2026-06-08
- **Status**: Proposed
- **Type**: Functional Requirement

## Description

The system must allow users to sign in using their Google account without needing to create a separate password for the application.

## Scenarios

- **Scenario 1: First-Time Google Sign-In**
  - **Given** a user is on the sign-in page,
  - **When** they click the "Sign in with Google" button,
  - **And** they successfully authenticate with their Google account,
  - **Then** the system automatically creates an associated account for them,
  - **And** they are granted access to the application.

- **Scenario 2: Returning Google Sign-In**
  - **Given** a user has previously signed in with their Google account,
  - **When** they click the "Sign in with Google" button again,
  - **Then** the system authenticates them and grants them access to the application.
