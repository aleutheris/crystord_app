# REQ-OR-260007: Suggested Label Source Uses Existing list_labels Query (Temporary)

- Class: OR
- Status: Active
- Date: 2026-05-08
- Source decision: Label suggestion source decision (temporary)

## Requirement

For the current implementation phase, the frontend shall source pre-search recommended labels from the existing backend `list_labels` query.

## Rationale

This provides an immediate, low-overhead source of real label data without introducing a new backend contract before the preferred per-user usage query is available.

## Scope Notes

- In scope for this requirement: use of existing `list_labels` as the temporary source for pre-search recommendations.
- Out of scope for this requirement: per-user most-used label query design and persistence model.

## Verification

- Integration check confirms recommendation retrieval uses `list_labels`.
- Runtime check confirms recommendations still appear before first search in blank-first entry.
- Contract check confirms no new backend query is required for this temporary phase.
